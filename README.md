# Text-Analysis

Spring과 Python을 사용한 텍스트 분석
===============================

* Mecab을 사용해 형태소분석을 하고 LSTM을 사용해 훈련을 하며 모델을 생성합니다.
* 훈련 결과를 Spring을 통해 시각적으로 제공합니다.
